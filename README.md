## Bike-sharing Dashboard
### Setup Environment
```
conda create --name main-ds python=3.9
conda active main-ds
pip install numpy pandas matplotlib seaborn babel streamlit
```

### How to Run Streamlit
```
streamlit run dashboard.py
```
